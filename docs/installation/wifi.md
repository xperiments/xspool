# Wi-Fi Setup

## Connecting xspool to Your Network

Once the installation is complete, follow these steps to set up Wi-Fi on your xspool device.

### Step 1: Connect to the xspool Hotspot

- A new Wi-Fi network will appear with the name **xspool-xxxx**, where **xxxx** is a unique identifier.
- Connect to this network using the following password:  
  **`12345678`**

### Step 2: Access the Wi-Fi Setup Page

- Open a web browser and go to:  
  **`http://192.168.4.1`**
- This page will allow you to scan for available Wi-Fi networks.

### Step 3: Select and Connect to Your Wi-Fi

- Choose your preferred Wi-Fi network from the list.
- Enter the Wi-Fi password when prompted.
- If the connection is unsuccessful, error messages will be displayed to help troubleshoot the issue.

### Step 4: Access xspool Locally

- Once connected to your Wi-Fi, you can access xspool by navigating to:  
  **`http://xspool-xxxx.local`** in your browser.

Now your xspool is connected and ready to use! 🚀
